# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an experimental implementation of a Model Context Protocol (MCP) server that enables Claude Code to connect to language servers like clangd. The goal is to bridge Claude Code with LSP (Language Server Protocol) servers to provide enhanced code intelligence capabilities.
